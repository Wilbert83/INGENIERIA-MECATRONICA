1.

Trigonometría

1.1 Definición de las funciones trigonométricas para un ángulo cualquiera

Razón: Es una comparación de dos magnitudes a través de un cociente.

Razones trigonométricas: Son las razones que existen entre los lados de un triángulo rectángulo y cambian al variar el ángulo en cuestión; es decir, están en función del ángulo.

Definición

Para un triángulo rectángulo con ángulo θ, las funciones trigonométricas relacionan los lados del triángulo (cateto opuesto, cateto adyacente e hipotenusa) con dicho ángulo.

Fig. 1 — Triángulo rectángulo y sus partes

1.2 Funciones trigonométricas para un ángulo agudo en un triángulo rectángulo

Las seis funciones trigonométricas principales se definen como:

sen(α) = cateto opuesto / hipotenusa
cos(α) = cateto adyacente / hipotenusa
tan(α) = cateto opuesto / cateto adyacente
Funciones trigonométricas básicas
Función Definición Recíproca
senoCO / Hipcsc(α)
cosenoCA / Hipsec(α)
tangenteCO / CAcot(α)
Recuerda que estas fórmulas sólo son válidas para ángulos agudos (0° < α < 90°) dentro de un triángulo rectángulo. La sección 1.3 extiende las definiciones a ángulos cualesquiera.
